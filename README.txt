Программа представляет собой простейщий файловый сервер.
Который ,при запросе из комадной строки Linux :
curl http://localhost:3000/static.js
,показывает содержимое данного файла.
при запросе несуществующего файла выводится ошибка.

также запрос можно сделать набрав
http://localhost:3000/static.js
в строке браузера.
